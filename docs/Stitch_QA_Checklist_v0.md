## 1. 固定检查原则

### 1.1 中文原则

- 面向用户可见的 UI 文案必须是简体中文。
- 允许保留：`Lumii`、`AI`、`kg`、`km`、合法人名/宠物名、品牌名。
- 不允许保留：`Discover`、`Profile`、`Messages`、`Retry Generation`、`Cancel` 这类通用 UI 英文。
- 图片 `alt`、`aria-label` 等无障碍文案后续也建议中文化，但优先级低于可见 UI。

### 1.2 乱码原则

任何疑似乱码都按 P0 处理，例如：
- `馛狗` -> `遛狗`
- `贅级` -> `超级`
- `匑猐`、`地嘖`、`抄恣` -> 对应导航文案

### 1.3 导航原则

主 Tab 固定为：

| Tab | 中文 |
|---|---|
| Pet | 宠物 |
| Nearby / Discover | 发现 |
| Map | 地图 |
| Messages | 消息 |
| Profile | 我的 |

规则：
- 主 Tab 页面需要底部导航。
- 登录、上传、AI 生成、错误页、编辑表单、记录体重等线性任务页不建议出现底部导航。
- 如果出现底部导航，文案、顺序、图标风格、active 态必须一致。
