# Lumii / 灵伴 视觉风格锚点 v0

版本：v0.1  
日期：2026-05-25

## 1. 品牌关键词

中文名：灵伴  
英文名：Lumii

关键词：

- 真实陪伴。
- 温暖但不幼稚。
- 有灵气。
- 轻社交。
- 可信赖的养宠助手。
- 宠物是主角，人是关系的连接者。

## 2. 电子宠物风格方向

主方向：真实卡通化。

解释：

- 不是完全写实照片。
- 不是过度幻想化的游戏宠物。
- 不是儿童玩具风的纯 Q 版。
- 应该像“真实宠物经过温柔卡通化后的数字分身”。

必须保留：

- 宠物真实毛色。
- 主要花纹和斑点。
- 耳朵形状。
- 眼睛颜色和神态。
- 体型比例。
- 明显特征，例如异瞳、短尾、卷毛、折耳、白手套、鼻尖颜色。

需要优化：

- 毛发边缘更干净。
- 表情更亲近。
- 轮廓更适合 App 内小尺寸展示。
- 色彩比真实照片略微明亮。
- 让电子宠物看起来可以“活在界面里”。

避免：

- 人类服装作为默认形象。
- 过度大眼睛、过度幼态。
- 站立成人形比例。
- 魔法、翅膀、盔甲等幻想元素。
- 与真实宠物特征无关的随机花纹。
- 看起来像通用贴纸包，而不是用户自己的宠物。

## 3. 样张方向

### A. 真实卡通头像

用途：

- 宠物档案头像。
- 附近宠物卡片。
- 聊天列表头像。

视觉要求：

- 正面或 3/4 视角。
- 背景简洁。
- 轮廓清楚。
- 小尺寸仍能看出宠物特征。

### B. 电子宠物全身形象

用途：

- App 首页电子宠物。
- 对话陪伴页。
- 健康提醒页。

视觉要求：

- 坐姿或自然站姿。
- 保留真实体型。
- 可以有轻微动态感，但不做夸张动作。
- 适合后续扩展表情、姿态和贴纸。

### C. 情绪表情

用途：

- 聊天贴纸。
- 健康状态反馈。
- 日记和提醒。

基础表情：

- 开心。
- 困困。
- 撒娇。
- 好奇。
- 委屈。
- 得意。

### D. 地图和社交卡片插画

用途：

- 空状态。
- 地图地点标记。
- 打招呼卡片。

视觉要求：

- 宠物和地点结合。
- 信息清楚，不抢主界面。
- 图形比电子宠物更简化。

## 6. App UI 视觉基调

整体 UI 应该更接近“安静、温暖、可信赖的日常工具”，而不是重游戏化宠物养成。

建议：

- 背景使用浅色中性色。
- 关键操作使用明亮但不过度刺激的品牌色。
- 电子宠物区域可以更有生命感。
- 健康管理页面要更清楚、更像工具。
- 社交页面要轻松，但不能像陌生人交友软件一样强刺激。

避免：

- 过重的糖果色。
- 大面积紫蓝渐变。
- 信息流过度娱乐化。
- 让用户误以为这是“虚拟宠物游戏”，而不是“真实宠物的数字伙伴”。
