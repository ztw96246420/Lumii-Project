# 账号注销执行与运营检查

日期：2026-07-11

## 1. 用户流程

1. 用户在“账号安全”发起注销，服务端向当前手机号发送一次性验证码。
2. 验证码确认成功后账号进入默认 30 天冷静期，服务端设置账号级 token 截止时间并撤销全部已记录登录会话；超出会话保留上限的旧 token 同样立即失效。App 清除本地账号状态并退出。
3. 冷静期内重新完成短信登录，视为用户主动撤销注销；原账号和业务数据继续使用。
4. 冷静期到期后，定时调度器和每次请求前置检查都会尝试执行永久清理，避免仅依赖单一 cron。
5. 到期清理后，相同手机号再次短信登录属于全新注册，历史宠物、内容和会话不会恢复。

## 2. 到期清理范围

- 用户资料、宠物档案、当前宠物、定位和收藏。
- 宠物日历中的体重、疫苗/驱虫、备忘和提醒。
- 灵伴原图、生成任务、动效任务、AI 对话、反馈样本和每日用量。
- 宠友圈小事、图片、评论、点赞、举报、拉黑、打招呼和约遛。
- 私信会话、消息以及其他用户侧关联通知。
- 地点点评、新增地点、贡献排行归属和地点贡献者手机号。
- Push 设备、站内通知、通知人群包、客服反馈、工单和质量复核。
- 用户处罚、申诉、待审批记录、登录失败状态、短信票据和手机号短信日用量。
- 移动端行为事件及明确以该手机号筛选生成的临时导出归档。

后台不可变审计日志按安全与合规留存策略保留，但到期执行新增的审计只记录匿名 HMAC 墓碑 ID、清理数量和时间，不记录手机号、昵称、宠物或内容。匿名墓碑只用于阻断注销前旧 token、证明删除已执行和关联对象存储清理任务，不能恢复账号。

## 3. 媒体文件销毁

- 清理前从用户关联状态中收集 COS object key，包括上传原图、灵伴结果图、动效视频和明确归属于该用户的临时导出。
- 业务状态先原子删除，object key 写入持久化销毁队列。
- COS `DELETE` 成功或对象已不存在（404）即完成；其他错误按指数退避重试，最长退避 24 小时。
- COS 临时不可用不会恢复用户业务数据，也不会丢失待删除任务。

## 4. 运行参数

| 环境变量 | 默认值 | 作用 |
| --- | ---: | --- |
| `ACCOUNT_DELETE_COOLING_OFF_MS` | 30 天 | 注销冷静期 |
| `ACCOUNT_DELETION_SWEEP_INTERVAL_MS` | 60 秒 | 到期账号与媒体队列扫描周期 |
| `ACCOUNT_DELETION_TOMBSTONE_RETENTION_MS` | 180 天 | 匿名墓碑保留期，实际不会短于 token TTL + 1 天 |
| `ACCOUNT_DELETION_OBJECT_DELETE_BATCH_SIZE` | 20 | 单次 COS 删除批量上限 |
| `ACCOUNT_DELETION_OBJECT_RETRY_BASE_MS` | 60 秒 | COS 删除失败首次重试间隔 |
| `AUTH_TOKEN_TTL_MS` | 30 天 | 登录 token 最长有效期 |

生产环境不得把冷静期配置为测试值。修改上述参数后，应重新执行账号注销专项测试和完整上线门禁。

## 5. 后台观察与处置

后台“系统健康”检查项 `account_deletion_processor` 展示：

- 冷静期内待删除账号数。
- 已超过计划删除时间但仍存在的账号数；大于 0 为发布阻断。
- COS 文件清理队列数量及发生过失败的数量。
- 匿名墓碑数量和最近一次实际删除时间。

上线台账同步包含 `account_lifecycle`。发现异常时：

1. 先确认后端进程、SQLite 写入和账号注销调度器正常。
2. 若账号逾期，检查服务日志中的 `Failed to process due account deletions`，恢复后由请求前置检查或下一次调度自动重试。
3. 若仅 COS 清理失败，检查 `COS_BUCKET`、`COS_REGION`、密钥和删除权限；业务数据已经删除，不应手工恢复账号。
4. 核对 `account.deletion.execute` 审计，不使用手机号反查匿名墓碑。

## 6. 自动化证据

`node scripts/smoke-account-deletion.cjs` 覆盖：

- 注销验证码错误与正确确认。
- 确认后全部旧会话失效。
- 超出服务端会话记录保留上限、已经从会话表裁剪的旧签名 token 也立即失效。
- 冷静期内短信登录自动撤销。
- 冷静期到期后账号与业务状态永久删除。
- COS 原图收到对应 `DELETE` 请求且持久化队列清空。
- 持久化状态不再包含已删除手机号。
- 注销前旧 token 永久失效，同手机号重新注册得到全新空账号。
