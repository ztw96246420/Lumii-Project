# 内容安全样本复审与误杀/漏杀回收

日期：2026-07-01

## 1. 目标

本次补齐内容安全后台里的样本治理闭环：

- 风险命中样本：关键词规则、腾讯云文本/图片机审命中后沉淀。
- 抽样复审样本：已通过内容按后台配置比例稳定抽样。
- 人工复审结论：确认风险、误杀、漏杀、忽略。
- 审计留痕：每次样本复审写入 `adminAuditLogs`。

样本复审是质量治理，不会自动改变用户内容状态。真正隐藏、删除、驳回、通过仍走内容安全任务池或图片审核池。

## 2. 后台配置

配置中心新增：

- `moderation.sampleReviewRatePercent`
- 取值范围：`0-100`
- 默认值：`0`

该配置只在后端生效，不下发到移动端 `/app/config`。

建议生产初期从 `1%-5%` 起步，先观察误杀/漏杀样本，再决定是否扩大抽样比例。

## 3. 样本来源

风险命中样本：

- 宠友圈小事文本。
- 宠友圈评论文本。
- 地点点评文本。
- 新增地点文本。
- 宠物资料公开文本。
- 宠物头像、AI 原图、宠友圈图片、宠友圈封面、工单附件等图片机审结果。

抽样复审样本：

- 上述公开内容通过内容安全规则或机审后，按 `sampleReviewRatePercent` 稳定抽样。
- 抽样不会影响用户侧发布、展示或通知。

## 4. 后台操作

内容安全页新增“内容安全样本复盘”卡片：

- 展示待复审、风险命中、抽样复审数量。
- 展示样本来源、Biztype、RequestId、风险标签、风险分、内容摘要和对象 ID。
- 支持就地标记：
  - `确认风险`
  - `误杀`
  - `漏杀`
  - `忽略`

接口：

- `POST /admin/moderation/samples/{sampleId}/review`

请求字段：

- `reviewStatus`: `confirmed | false_positive | false_negative | ignored`
- `reason`: 复审说明

## 5. 移动端联动

移动端不会读取抽样率，也不会看到样本复审结论。

联动边界：

- 内容处理动作仍由审核任务池控制，影响 App 可见性。
- 样本复审只沉淀策略质量信号，用于后续调关键词、Biztype 策略、腾讯云内容安全配置和人工审核 SOP。
